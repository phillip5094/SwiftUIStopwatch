# SwiftUIStopwatch

SwiftUI 기반 iOS, watchOS, macOS에서 동작하는 Stopwatch 앱 입니다. 

## Requirments

* Xcode 13.1
* iOS 15.0
* watchOS 8.0
* macOS 11.5
